A few things to remember...

html tag 1: td
Td represents a table data cell, imperative tag for tables.
html tag 2: img
This tag allows you to attach pictures to your html website.
html tag 3: a href
This tag allows you to anchor a link and add the name for the link that will show up on the web page.
html tag 4: i
this tag aloows you to italicize text.
html tag 5: mark
Mark will highlight relevant text when needed.
